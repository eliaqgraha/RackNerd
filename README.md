## RackNerd 介绍
RackNerd是一家成立于2019年的美国主机商，主要提供美国机房的虚拟主机、VPS和独立服务器产品。在美国和欧洲均设有数据中心，特别在洛杉矶、圣何塞、西雅图等地提供服务，适合国内用户使用。
